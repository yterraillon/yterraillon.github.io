1. Données collectées

Données actuellement collectées

Actuellement, En Bref ne collecte aucune donnée personnelle. L’application fonctionne sans nécessiter d’inscription, de suivi, ou de collecte d’informations identifiables.

Évolutions prévues

Certaines fonctionnalités futures pourront nécessiter la collecte limitée de données, par exemple :

  • Paiement via l’App Store : géré directement par Apple. En Bref ne traite ni ne stocke les données de paiement.
  • Stockage iCloud : utilisé uniquement pour synchroniser vos préférences ou votre historique de lecture, si cette fonction est activée.
  • Code promotionnel : il pourra être lié à un identifiant unique de votre appareil afin d’éviter les abus. Cet identifiant ne permet pas de vous identifier personnellement, mais pourra être stocké de manière sécurisée pour valider votre accès.

2. Utilisation des données

Si des données sont collectées dans le futur, elles le seront uniquement pour :

  • Fournir les fonctionnalités prévues de l'application.
  • Prévenir la fraude (ex. limitation d’un code promo à un seul appareil).
  • Améliorer l’expérience utilisateur.

Aucune donnée ne sera utilisée à des fins publicitaires, ni partagée avec des tiers sans votre consentement.

3. Sécurité

Nous mettons en œuvre des mesures de sécurité raisonnables pour protéger vos données, si collectées. En particulier :

  • Les éventuelles données locales seront stockées de manière sécurisée sur votre appareil.
  • Aucune donnée n’est transmise à des serveurs tiers sans nécessité ou votre accord.

4. Vos droits

Conformément au Règlement général sur la protection des données (RGPD) :

  • Vous pouvez demander l’accès, la rectification ou la suppression de vos données si l’application en collecte dans le futur.
  • Vous pouvez désactiver iCloud à tout moment via les réglages de votre appareil.
